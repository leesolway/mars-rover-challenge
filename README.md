#### Getting Started

**Install dependencies**
`npm install`

**Instructions**

Add instructions to the instructions.txt file, following the example below

```
5 5
1 2 N
LMLMLMLMM
3 3 E
MMRMMRMRRM
```

**Run tests**
`npm test`

**Run program**
`npm start`
